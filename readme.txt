Создадим структуру text, которая будет содержать строковый вектор и несколько методов, позволяющих смоделировать описанный в условии
задачи процесс (например метод arase удаляет последний элемент той строки, на которой находится курсор). Таким образом элементы вектора структуры будут имитировать строки,
которые пользователь импровизированного редактора видит на экране. При обработке каждого символа будем находить максимальную длину среди всех строк.
Полученное после обработки всех символов значение и будет ответом на задачу.
